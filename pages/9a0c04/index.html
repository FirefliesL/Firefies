window的显示

在DisplayPolicy中，有一个layoutWindowLw方法，这个方法中会计算window的位置，
计算出WindowState中各种条件下的显示区域：

  • Rect pf = windowFrames.mParentFrame 通常表示整个屏幕的区域，即整个屏幕的大小。但可能需要将其名称与mDisplayFrame互换，以提高代码的清晰性。
  • Rect df = windowFrames.mDisplayFrame 表示ActivityStack中的窗口所在的整个屏幕区域。通常与设备的屏幕区域相等。
  • Rect cf = windowFrames.mContentFrame 当IME（如键盘）未显示时，等于mDecorFrame。当IME显示时，等于mDecorFrame减去IME占用的区域。
  • Rect vf = windowFrames.mVisibleFrame 用于处理一些旧版应用程序的IME（输入法）显示提示文本的情况。即用于处理兼容旧版应用的情况。
  • Rect dcf = windowFrames.mDecorFrame 表示除去状态栏和导航栏之外的屏幕区域。如果同时显示状态栏和导航栏，则mDecorFrame等于mStableFrame。
  • Rect sf = windowFrames.mStableFrame 表示除去稳定不变的内边距之后的mDisplayFrame。该值始终保持不变，无论状态栏或导航栏是否可见。
