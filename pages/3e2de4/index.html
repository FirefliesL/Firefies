AndroidStudio IBUS中文输入异常

ubuntu下使用默认IBUS输入法，在AndroidStudio中输入中文时，不显示待选中文，输入后一直显示一条下划线

  1. 点击菜单 “Help | Edit Custom VM options…”
  2. 添加 -Drecreate.x11.input.method=true 到最后一行
  3. 重启IDEA

Tools–Create Desktop Entry

修改默认Project视图

要将默认视图从 Android 视图更改为 Project 视图，请依次选择 Help > Edit Custom Properties 并添加 studio.projectview=true。

project structure

  1. File – Project Structure (Ctrl alt shift s)
  2. 项目右键 – Open Module Settings （F4）

定制toolbar

File – Settings – Appearance & Behavior – Menus and Toolbars

每次都下载gradle

配置环境变量 GRADLE_HOME ， GRADLE_USER_HOME

  • GRADLE_HOME
    Gradle 的安装目录。可以用来指定本地 Gradle 版本，而不是使用包装器。
  • GRADLE_USER_HOME
    Gradle 存储其全局配置属性、初始化脚本、缓存、日志文件等的目录。若未设置，则默认为USER_HOME/.gradle
    jar都会下载到GRADLE_USER_HOME/caches/modules-2/files-2.1下面

在android studio中指定gradle user home
第一次可以配置代理下载
