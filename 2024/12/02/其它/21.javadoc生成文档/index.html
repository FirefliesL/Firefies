发表于2024-12-02|更新于2025-01-14
|阅读量:

简单使用

简单的生成：
如有一个Test.java文件，-d指定生成doc的目录为OutDoc

还可以通过-sourcepath指定源码目录

一般都会有依赖，可以通过 -classpath指定依赖的jar的路径

不为过时接口生成文档

-nodeprecated

过时接口生成文档，但不显示“过时”标记或列表，也不会列出过时的 API 链接

-nodeprecatedlist
-windowtitle 文档的浏览器窗口标题
-nohelp 不生成帮助页
-noindex 不生成索引
-notree 不生成类分层结构
-doctitle 概览页面的标题
-notimestamp 不生成时间戳注释
